## 개발 환경

* **Framewrok** : Node.js(>=18.18.0), Next.js 14
* **Language** : TypeScript

## 제공 API
* [Nager.Date](https://date.nager.at/Api)


## 시작하기
 
### 1. 저장소 클론
~~~sh
git clone https://github.com/yoonucho/yu-calendar.git
~~~

### 2. node.js 설치(>=18.18.0)
https://nodejs.org/en/download


### 3. 모듈 설치 & 실행 

~~~sh
npm install
npm run dev
~~~
